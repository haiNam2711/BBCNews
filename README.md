# App BBC News

## Mô tả:
App đọc báo sử dụng ngôn ngữ Swift trên hệ điều hành iOS, trong đó lấy dữ liệu từ NewsAPI để xuất ra các bài báo, lưu trữ các bài báo vào Realm Database
